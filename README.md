# Tarea_flujo-de-control

# Resumen
Este programa permite ingresar el sueldo de un empleado en República Dominicana y calcula cuánto debe pagar de ISR (Impuesto Sobre la Renta) según la escala salarial vigente publicada por la Dirección General de Impuestos Internos (DGII) para el año actual.

El sistema verifica si el sueldo está dentro del rango exento; si es así, muestra “N/A” porque no debe pagar impuesto. Si el sueldo supera ese límite, aplica el porcentaje correspondiente según el tramo establecido y muestra en pantalla tanto el sueldo como el monto que debe pagar por ISR.

El objetivo del programa es facilitar el cálculo del impuesto de manera clara, rápida y automática.
